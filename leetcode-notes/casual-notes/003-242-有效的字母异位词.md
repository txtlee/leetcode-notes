# 有效的字母异位词

## 题目
给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的 字母异位词。

 

示例 1:

输入: s = "anagram", t = "nagaram"
输出: true
示例 2:

输入: s = "rat", t = "car"
输出: false

## 解题思路
tip1：先比对两个字符串长度，不一致直接返回 False
tip2：将字符串 s 的不同字母及其出现次数记录到哈希表中，然后遍历字符串 t 的每个字母，每个字母对应的哈希表的  value - 1，如果某次减过之后小于 0 了直接返回 False
tip3：这类题除了用哈希表，还可以用数组，26个字母的话数组长度就是26，更多字母的话依次类推，遍历字符串 s 每次 + 1，遍历字符串 t 每次 - 1，遇到值小于 0 时直接返回 False

## 代码
class Solution(object):
    def isAnagram(self, s, t):
        if len(s) != len(t):
            return False
        hashtable_s = dict()
        hashtable_t = dict()
        for i in range(len(s)):
            ch_s = s[i]
            ch_t = t[i]
            if ch_s not in hashtable_s.keys():
                hashtable_s[ch_s] = 0
            hashtable_s[ch_s] = hashtable_s[ch_s] + 1
            if ch_t not in hashtable_t.keys():
                hashtable_t[ch_t] = 0
            hashtable_t[ch_t] = hashtable_t[ch_t] + 1
        for key, value in hashtable_s.items():
            # if hashtable_t[key] != value:
            """
            hashtable_t[key] 这种写法,如果 t 里根本没有这个 key 会直接报 KeyError。比如 s
  有 'a' 但 t 里没有 'a',程序不是"返回 False",而是直接崩。
            """
            if hashtable_t.get(key, 0) != value:
                return False
        
        return True
