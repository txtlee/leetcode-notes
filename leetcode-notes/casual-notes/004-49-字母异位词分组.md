# 字母异位词分组

## 题目
给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

 

示例 1:

输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]

输出: [["bat"],["nat","tan"],["ate","eat","tea"]]

解释：

在 strs 中没有字符串可以通过重新排列来形成 "bat"。
字符串 "nat" 和 "tan" 是字母异位词，因为它们可以重新排列以形成彼此。
字符串 "ate" ，"eat" 和 "tea" 是字母异位词，因为它们可以重新排列以形成彼此。
示例 2:

输入: strs = [""]

输出: [[""]]

示例 3:

输入: strs = ["a"]

输出: [["a"]]

## 解题思路对每个字符串进行排序，这样所有的字母异位词直接对比是否一致即可
**思路1 - 排序**
1. 遍历字符串列表
2. 将当前字符串按照字母顺序排序
3. 将排序后的字符串作为 key 去查哈希表，不存在的话新增空列表，然后将排序前的字符串 append 进去
4. 遍历哈希表的 values()，将每个 value 取出来放到返回体的列表中
**思路2 - 计数**
和思路 1 近似，只是哈希的 key 改为 26 个小写字母的出现次数统计。注意，列表不能作 key，但是将列表转化为 tuple 可以做 key。


## 代码
class Solution(object):
    def groupAnagrams(self, strs):
        """
        :type strs: List[str]
        :rtype: List[List[str]]
        """
        hashmap = dict()
        for str in strs:
            sort_str = "".join(sorted(str))
            if sort_str not in hashmap:
                hashmap[sort_str] = []
            hashmap[sort_str].append(str)
        
        res_lst = []
        for val in hashmap.values():
            res_lst.append(val)
        
        return res_lst
